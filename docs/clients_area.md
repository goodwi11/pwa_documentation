# Личный кабинет

## Обзор интерфейса

После входа в систему вы попадаете в личный кабинет. 

Интерфейс состоит из:

**Левое меню:**
- **Создать PWA** — кнопка для создания нового приложения
- **Главная** — список всех ваших PWA
- **Конструктор** — редактор для создания и изменения PWA
- **Статистика** — аналитика по вашим приложениям
- **Пуши** — управление push-уведомлениями

**Верхнее меню:**
- Личный кабинет
- Pricing — тарифы
- FAQ — частые вопросы
- Documentation — документация
- Переключатель языка
- Clients Area — вход/выход

## Список PWA-приложений

На главной странице отображается таблица со всеми вашими PWA-приложениями. Для каждого приложения показаны:

| Колонка | Описание |
|---------|----------|
| Название | Имя вашего PWA |
| ID | Уникальный идентификатор приложения |
| Домен | Привязанный домен |
| Язык | Язык интерфейса приложения |
| Создан | Дата создания |
| Изменён | Дата последнего изменения |
| Статус | Текущий статус |
| Действие | Кнопки редактирования и удаления |

## Статусы PWA

Каждое PWA может находиться в одном из статусов:

- **Активный** (зелёный) — приложение опубликовано и доступно по указанному домену, фильтрация трафика активна.
- **Вайт** (жёлтый) — приложение опубликовано и доступно по указанному домену, переход по домену направляет весь трафик на вайт.
- **Контент** (красный) - приложение опубликовано и доступно по указанному домену, переход по домену направляет весь трафик на PWA.
- **Черновик** (синий) — процесс созданиия PWA не завершен.

## Управление готовыми PWA

Для каждого созданного PWA в таблице доступны действия:

- **Редактировать** (иконка карандаша) — открывает конструктор для изменения настроек PWA
- **Удалить** (иконка корзины) — удаляет приложение без возможности восстановления

!!! warning "Внимание"
    Удаление приложения необратимо. Все данные, включая статистику и привязку домена, будут потеряны.
