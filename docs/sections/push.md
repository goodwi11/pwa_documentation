# Push-уведомления

Push-уведомления — мощный инструмент для возврата и повышения удержания пользовтаелей. В Comsign PWA встроена полноценная система создания и управления пушами.

## Обзор раздела

Раздел **«Пуши»** в левом меню открывает страницу управления и создания ваших push-уведомлений.<br>

**Таблица пушей:**

| Колонка | Описание |
|---------|----------|
| Название | Имя пуш-уведомления |
| Тип | По расписанию / По событию |
| Событие | Триггер отправки |
| Создан | Дата создания |
| Изменён | Дата последнего изменения |
| Статус | Текущий статус |
| Действие | Редактирование, копирование, удаление |

Фильтры:<br>
- **Все** — все созданные пуши<br>
- **Активные** — пуши, которые отправляются<br>
- **Отправленные** — уже отправленные уведомления<br>
- **Черновики** — сохранённые, но не активные<br>
- **Приостановленные** — временно остановленные пуши

## Создание пуша

Для создания нового пуш-уведомления нажмите **«Создать пуш»**. Откроется форма с настройками.

**Шаг 1. Основные настройки**

**Название**<br>
Внутреннее название пуша для вашего удобства. Пользователи его не видят.

**Вид**<br>
Выберите тип отправки и настройте параметры отправки:

- **По расписанию** — пуш отправляется в указанное время
- **По событию** — пуш отправляется при срабатывании триггера

**Шаг 2.** Настройка контента и оформления

**Заголовок**<br>
Основной текст уведомления, который виден в шторке. Рекомендуемая длина — до 50 символов.

**Иконка**<br>
Загрузите иконку пуша (отображается слева от текста). Формат: JPEG/PNG, рекомендуемый размер: 192x192 px.

**Картинка**<br>
Большое изображение, которое отображается в развёрнутом уведомлении. Формат: JPEG/PNG.

**Текст**<br>
Дополнительный текст под заголовком. Здесь можно дать больше информации или call-to-action.

**Ссылка**<br>
URL на который перейдёт пользователь при клике на пуш. Если оставить пустым, откроется оффер из настроек PWA.

**Языки подписчиков**<br>
Выберите, пользователям с какими языковыми настройками отправлять пуш.<br>
Это позволяет создавать разные версии пушей для разных ГЕО и показывать релевантный контент.

!!! note "Примечание"
    Справа от формы отображается превью — так ваш пуш будет выглядеть на устройстве пользователя.

После заполнения всех полей нажмите **«Сохранить»**. Созданный пуш появится в списке и станет доступен для выбора в конструкторе PWA.
