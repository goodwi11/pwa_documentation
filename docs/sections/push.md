# Push-уведомления

Push-уведомления — мощный инструмент для возврата пользователей и повышения retention. В Comsign PWA встроена полноценная система создания и управления пушами.

## Обзор раздела

Раздел **«Пуши»** в левом меню открывает страницу управления и создания ваших push-уведомлений.

**Структура раздела:**

Над таблицей расположены фильтры:
- **Все** — все созданные пуши
- **Активные** — пуши, которые отправляются
- **Отправленные** — уже отправленные уведомления
- **Черновики** — сохранённые, но не активные
- **Приостановленные** — временно остановленные пуши

В правом верхнем углу — кнопка **«Создать пуш»**.

**Таблица пушей:**

| Колонка | Описание |
|---------|----------|
| Название | Имя пуш-уведомления |
| Тип | По расписанию / По событию |
| Событие | Триггер отправки |
| Создан | Дата создания |
| Изменён | Дата последнего изменения |
| Статус | Текущий статус |
| Действие | Редактирование, копирование, удаление |

## Создание пуша

Для создания нового пуш-уведомления нажмите **«Создать пуш»**. Откроется форма с настройками.

### Основные настройки

**Название**
Внутреннее название пуша для вашего удобства. Пользователи его не видят.

**Вид**
Выберите тип отправки:
- **По расписанию** — пуш отправляется в указанное время
- **По событию** — пуш отправляется при срабатывании триггера

## Типы отправки

### По событию

При выборе этого типа укажите:

**Событие для отправки**
Выберите триггер из списка:
- **Инстал** — пуш отправится сразу после установки PWA

**Задержка**
Укажите задержку перед отправкой после срабатывания события:
- Дней: 0-30
- Часов: 0-23
- Минут: 0-59
- Секунд: 0-59

Например, задержка «0 дней, 1 час, 0 минут, 0 секунд» означает, что пуш уйдёт через час после установки.

### По расписанию

При выборе этого типа пуш будет отправлен всем пользователям в указанное время. Настройте дату и время отправки в соответствующих полях.

## Настройка контента и оформления

### Контент и оформление

**Заголовок**
Основной текст уведомления, который виден в шторке. Рекомендуемая длина — до 50 символов.

**Иконка**
Загрузите иконку пуша (отображается слева от текста). Формат: JPEG/PNG, рекомендуемый размер: 192x192 px.

**Картинка**
Большое изображение, которое отображается в развёрнутом уведомлении. Формат: JPEG/PNG.

**Текст**
Дополнительный текст под заголовком. Здесь можно дать больше информации или call-to-action.

**Ссылка**
URL, на который перейдёт пользователь при клике на пуш. Если оставить пустым, откроется оффер из настроек PWA.

### Языки подписчиков

Выберите, пользователям с какими языковыми настройками отправлять этот пуш:
- RU — Россия
- IT — Италия
- DE - Германия
- И другие доступные локализации

Это позволяет создавать разные версии пушей для разных ГЕО и показывать релевантный контент.

### Превью

Справа от формы отображается превью — так ваш пуш будет выглядеть на устройстве пользователя.

После заполнения всех полей нажмите **«Сохранить»**. Созданный пуш появится в списке и станет доступен для выбора в конструкторе PWA.
