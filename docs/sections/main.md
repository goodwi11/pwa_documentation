Главная
=======

Данный раздел показывает список всех ваших PWA и основную статистику по каждому из них. Здесь вы можете быстро найти нужное PWA, отфильтровать по параметрам и перейти к редактированию или анализу.

Элементы интерфейса
-------------------

**Список приложений:**
Для каждого приложения отображаются ключевые параметры (например: название, домен, статус, дата создания и т.д.).

**Фильтры:**
- Название — поиск по названию приложения.
- ID — поиск по внутреннему идентификатору приложения.
- Домен — фильтрация по домену, на котором опубликовано приложение.
- Язык — фильтрация по языку приложения (например, ru, en).
- Создан — фильтрация по дате создания.
- Изменен — фильтрация по дате последнего изменения.
- Статус — фильтрация по текущему состоянию (активно, черновик и т.д.).
- Действие — быстрый доступ к операциям (редактировать, открыть, статистика и т.п.).

**Вкладки статуса сверху:**
- Все — отображаются все приложения без ограничений по статусу.
- Активный — только опубликованные и работающие приложения.
- Вайт — приложения, используемые как white (вайты) в клоакинге.
- Контент — приложения контентного типа (в зависимости от логики сервиса).
- Черновик — приложения, которые ещё не опубликованы и находятся в работе.
